Usage: snail project update [options]

update a project's metadata

Options:
  -p, --project <domain>       specify which project
  --domain <new_domain>        set new domain
  --description <description>  set description
  --private                    set legacy private flag (deprecated)
  --no-private                 clear legacy private flag (deprecated)
  --privacy <privacy>          set privacy (public, private_code, or private_project)
  -h, --help                   display help for command
